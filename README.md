# 🔐 Secret Code Generator (Bash)

A simple Bash script to generate secure random secret codes.

## 🚀 Features
- Custom length input
- Uses:
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Symbols
- Save generated codes to file
- Fast & lightweight
- Terminal-based

## 📦 Installation

```bash
git clone https://github.com/gowrang-kuthurvad/secret-code-generator.git
cd secret-code-generator
chmod +x generator.sh
